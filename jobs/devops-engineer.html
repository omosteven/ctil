DevOps Engineer

Workplace: Warsaw
Apply for a job

Please include the following statement in your CV :
I agree to the processing of the data contained in the application, which is necessary for the recruitment
process (Journal of Laws of 1997, No. 133, item 883).

Requirements
  • Experience when working with Linux and its distributions (CentOS/RHEL, Debian/Ubuntu)
  • Automation of building and deployment: Jenkins, Bamboo
  • Docker experience, especially containerization of Java applications
  • Small-scale container management with the use of Docker Compose or Docker Swarm
  • Large-scale container orchestration experience with the use of Kubernetes and Helm
  • Experience in working with cloud resources (Google Cloud, AWS, Azure)
  • Knowledge of how to work with Git code repositories
  • Implementing the required solutions while following the KISS principle – Keep It Simple Stupid – in order to avoid excessively complicated management in the future
Responsibilities
  • Support for production and test environments
  • Cooperation with software engineers
  • Configuration, optimization, monitoring and maintenance of servers
  • Administering and troubleshooting Linux-based systems
  • Problem solving across a wide range of services and functional areas
  • Carrying out analyzes and optimization of infrastructure costs
Advantages
  • Automation of the infrastructure administration (Terraform, Ansible)
  • Java development experience, especially managing project lifecycles (Maven, Gradle)
  • NPM-based project experience (for example Angular), especially managing project lifecycles
  • Flexibility in creating highly available, durable and scalable infrastructure both on premises and in the cloud – and also how to recreate cloud-provided solutions on premises, for example by using Grafana Loki for central logging
  • Securing the infrastructure – proper firewall configuration, minimum required permissions etc
  • Knowledge of database administration (PostgreSQL, SQL Server, MySQL)
We offer
  • Continuity of projects
  • Flexible working hours and freedom in implementation
  • Attractive salary proportional to the skill level
  • Convenient access, no parking problems, the possibility of working remotely
  • Multisport card
  • Medical package
  • Fresh fruit in the office
  • Physiotherapist
  • English classes
